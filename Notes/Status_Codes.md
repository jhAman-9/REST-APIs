# Status code      Range = (100 --> 500)

- 1XX -> Informational Code

- 2XX -> Success Code

    - 201 -> Created
    - 204 -> No Content

- 3XX -> Readirection Code

    - 302 -> Found
    - 304 -> Not Medified
    - 307 -> Temporary Redirect


 - 4XX -> Client Error Code

    - 400 -> Bad Request
    - 401 -> Unauthorized
    - 403 -> Forbidden 
    - 404 -> Not Found
    - 415 -> Unsupported Media Type

 - 5XX -> Server Error Code

    - 500 -> Internal Server Error
